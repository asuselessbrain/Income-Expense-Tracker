# 🚀 Quick Start Guide - FinTrack Enhanced

Welcome to FinTrack! Here's how to get started with the new professional features.

## 5-Minute Setup

### 1. **Add Your First Transaction** (Already works)
- Click the **+** button or go to **Add**
- Enter amount, select category, type, and notes
- Transactions appear immediately on Dashboard

### 2. **Set Monthly Budgets** ⭐ NEW
- Open sidebar → Click **Budgets**
- Click "Add Budget"
- Select a category, enter limit (e.g., Food: $300/month)
- Watch your spending tracked in real-time with color warnings
- Already have sample budgets loaded!

### 3. **Create Financial Goals** 🎯 NEW
- Open sidebar → Click **Goals**
- Click "Add Goal"
- Example: "Emergency Fund" target $5,000, deadline Dec 2024
- Track your progress as you save
- Get notified when complete!

### 4. **Check Smart Insights** 💡 NEW
- Go to **Dashboard**
- Scroll below Summary Cards
- Read automated intelligence about your spending
- Get recommendations on what to do

### 5. **Explore Your Data**
- **Analytics**: Detailed charts comparing months/years
- **Categories**: Manage how transactions are organized
- **Settings**: Export data, change theme, set currency

## 🎯 Common Tasks

### Set Up Monthly Budget
1. **Budgets** → **Add Budget**
2. Select **Food** category
3. Enter **$400** limit
4. Leave month/year as current
5. Click **Create**
6. As month goes on:
   - Green (0-80%): Safe
   - Yellow (80%+): Caution warning ⚠️
   - Red (100%+): Over budget 🚨

### Create a Savings Goal
1. **Goals** → **Add Goal**
2. Title: "Vacation Fund"
3. Target: $2,000
4. Already saved: $500
5. Select deadline: June 30, 2024
6. Pick icon: **Plane** ✈️
7. Click **Create**
8. Progress bar shows: 25% (500/2000)
9. Every transaction updates goal page

### Export Your Data
1. **Settings** → Scroll to "Data Management"
2. Click **"Export CSV"** for all transactions
3. Or **"Export Summary"** for overview with totals
4. File downloads automatically
5. Open in Excel/Google Sheets/Numbers

### Track Spending Patterns
1. Open **Analytics**
2. Use filter:
   - **Monthly**: This month vs last month
   - **Yearly**: This year vs last year
   - **Custom**: Pick exact date range
3. See:
   - Spending comparison with % change
   - Top spending categories
   - Income vs expense trends
   - Savings rate percentage

### Customize Your Categories
1. **Categories** in sidebar
2. Toggle between **Income** / **Expense** tabs
3. See default categories (can't edit)
4. Click **Add Category** to create custom one
5. Example: "Subscriptions" category
   - Type: Expense
   - Icon: Music note 🎵
   - Color: Purple
6. Now use in transactions!

## 📊 Understanding New Features

### Budget System
**What**: Set spending limits per category per month
**Why**: Control your spending and get warnings
**Where**: Navigate to Budgets
**Result**: Visual progress bars + alerts when over

### Goals System
**What**: Save money toward specific targets with deadlines
**Why**: Motivates you, tracks progress, celebrates wins
**Where**: Navigate to Goals  
**Result**: Track multiple goals, see percentage complete

### Smart Insights
**What**: AI-based recommendations about your spending
**Why**: Automatically spot issues and opportunities
**Where**: Dashboard (automatically shown)
**Result**: "You spent 25% more this month" suggestions

### Advanced Filters
**What**: See transactions for Monthly / Yearly / Custom date ranges
**Why**: Compare different periods
**Where**: Buttons at top of Dashboard, Analytics
**Result**: All charts and exports use selected filter

### Category Management
**What**: Create custom income/expense categories
**Why**: Better organization + use custom categories in transactions
**Where**: Categories page in sidebar
**Result**: Custom categories available in transaction form

### Data Export
**What**: Download all data as CSV files
**Why**: Backup, analyze in Excel, import elsewhere
**Where**: Settings → Data Management
**Result**: CSV files you can open/share/analyze anywhere

## 💡 Pro Tips

### 1. Set Realistic Budgets
- Review spending from last 3 months
- Set budget 10-15% lower than average
- Gradually improve over time

### 2. Use Multiple Goals
- Create 3-5 goals: Emergency, Vacation, Car, House, etc.
- Saves feel more purposeful
- Dashboard shows overall progress

### 3. Check Insights Weekly
- Smart Insights automatically updates
- Catch spending issues early
- Act on recommendations

### 4. Organize Categories
- Start with defaults
- Create 3-5 custom categories for YOUR needs
- Keep ~15-20 total categories

### 5. Review Analytics Monthly
- Compare month vs month
- Identify top spending
- Celebrate improvements

### 6. Export Before Important Events
- Export before taxes
- Export for financial advisor
- Keep backup of data

## 🆘 Troubleshooting

### Budget showing incomplete
✓ Make sure category matches exactly
✓ Check month/year is correct
✓ Ensure transactions are assigned to category

### Goal shows "Overdue"
✓ That's correct if deadline is in past
✓ Mark as complete to stop showing as overdue

### Insights not appearing  
✓ Need transactions to analyze
✓ Insights update automatically when transactions added
✓ Try adding a test transaction

### Data not exporting
✓ Check filters aren't hiding data
✓ Must have transactions to export
✓ Use current window to download file

### Wrong currency showing
✓ Go to Settings → Currency
✓ Select your preferred currency
✓ All values update immediately
